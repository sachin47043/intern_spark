# 💼 Personal Portfolio Website

A clean and responsive personal portfolio website built using **HTML, CSS, and JavaScript**. This project showcases my basic frontend development skills and serves as my personal portfolio.

## 📌 Overview

This portfolio includes:

- Responsive Navigation Bar
- Hero Section with Typing Animation
- About My Skills & Experience
- Work Experience Timeline
- Simple Footer Section
- Clean and Modern UI

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript
- Typed.js
